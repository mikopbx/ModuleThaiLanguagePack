# ModuleThaiLanguagePack

Complete Thai language pack for MikoPBX including UI translations and voice prompts.

## What's Included

- **Voice Prompts**: Thai voice prompts for system menus, greetings, and notifications
- **UI Translations**: Complete Thai translation of MikoPBX admin interface

## Installation

1. Download and install the module from MikoPBX Marketplace
2. Enable the module in **Modules** section
3. Go to **General Settings** and select Thai (ไทย) as the system language

## Requirements

- MikoPBX 2025.1.1 or later
